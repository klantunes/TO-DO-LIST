# 📝 TO DO LIST

Projeto de uma lista de tarefas desenvolvido para praticar JavaScript, manipulação do DOM e criação de interfaces interativas.

A ideia do projeto foi criar uma To Do List simples, mas com funcionalidades mais completas e uma interface moderna, focando principalmente em melhorar minhas habilidades no front-end.

## Funcionalidades

* Adicionar tarefas
* Editar tarefas
* Remover tarefas
* Marcar tarefas como concluídas
* Filtros de tarefas
* Barra de progresso
* Dark mode
* Salvamento automático com LocalStorage
* Toast de notificações
* Layout responsivo

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Font Awesome

## O que pratiquei nesse projeto

Durante o desenvolvimento eu pratiquei bastante:

* Manipulação do DOM
* Eventos no JavaScript
* LocalStorage
* Responsividade
* Animações com CSS
* Organização de código
* Lógica de programação
